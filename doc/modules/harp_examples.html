Package harp-examples

@here/harp-examples

Overview

This repository contains examples for harp.gl.

Building and running

You can build and run the examples running the following commands from the root path of the project:

yarn install
yarn run build
yarn start

Open http://localhost:8080 in a web browser to try the examples.

Examples

We can group the examples by several categories, looking at what kind of features and capabilities they showcase.

Getting Started

  1. The hello world example that displays a map in an HTML element with our default map style and default data source.
  2. The hello add simple object example that shows how to add a THREE.js object to the map.

Camera

  1. Camera with free movement, that demonstrates how the camera works in harp.gl.

Text Rendering

  1. Showcase of the dynamic text rendering capabilities of our text rendering library.

Data sources

  1. Display an interactive map of Italy on a reduced map style that showcases picking and using GeoJSON Data that relies on geojson-datasource.
  2. Show OMV Data with our default map style.
  3. Render raster map tiles using the webtile-datasource.
  4. Show OSM MVT Data with our default map style.

Styling

  1. Display three map views, side by side, in which we show the a map with three different styles at the same time, using OMV Data.
